# 🚀 HireFlow AI - Recruitment Portal

HireFlow AI is a Full Stack Java Recruitment Portal developed using Spring Boot, React, JWT Authentication, and MySQL.

## 📌 Features

### 👨‍💼 Candidate
- User Registration & Login
- JWT Authentication
- Search Jobs
- Apply for Jobs
- Upload Resume
- Save Jobs
- View My Applications
- Update Profile

### 🧑‍💼 Recruiter
- Recruiter Login
- Post Jobs
- Edit Jobs
- Delete Jobs
- View Applicants
- Download Candidate Resumes
- Shortlist / Reject Candidates

### 👑 Admin
- Dashboard Statistics
- Manage Users
- Manage Jobs
- View Applications
